This is a simple frontend web application built as a **Proof of Concept (PoC)** for the **AI Subscription Guard** solution. It helps users manually track recurring monthly payments and calculate the total spend, addressing the core problem of "Subscription Fragmentation" mentioned in the project proposal.

---

## üöÄ Key Features

* **Add Subscription:** Allows users to input the name and monthly cost of a new subscription.
* **Cost Analysis:** Calculates and displays the **Total Monthly Spend** in real-time.
* **Removal:** Users can easily remove subscriptions from the tracker.
* **Simple UI:** Clean, responsive interface for easy tracking.
